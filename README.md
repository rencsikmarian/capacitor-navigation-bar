# capacitor-navigation-bar

Navigation Bar plugin for Capacitor

## Install

To use npm

```bash
npm install capacitor-navigation-bar
````

To use yarn

```bash
yarn add capacitor-navigation-bar
```

Sync native files

```bash
npx cap sync
```
